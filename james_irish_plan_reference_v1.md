## Summer plan — 6 sessions (22 June – end August 2026)

No fixed day. Slot into schedule when a gap appears. Roughly one session every 1–1.5 weeks. Log each at row-fysk.vercel.app/study.html.

**Session ordering:** S1 must happen before S2 and S6 (phrase bank and topics must be chosen first). S3 must happen before S6 (timing strategy must be built before second RC attempt). S4 and S5 are fully independent. All 6 sessions can float in any order.

### Session 1 — Essay Phrases + Topic Selection (~50 min)
**Resources:** Grinds360 (Universal Essay Phrases content); Pointsgame.ie

1. Run Pointsgame.ie frequency analysis for HL Irish essay topics; choose top 2
2. Extract key sections from Grinds360 Universal Essay Phrases: generic intro, conclusion, connectors, conditional mood phrases; write personal phrase bank
3. Find sample intro and conclusion in Grinds360 essay content; rewrite in your own Irish — same structure, your phrasing

**Target:** Top 2 topics chosen; phrase bank built; intro/conclusion formula owned

### Session 2 — First Timed Essay + Pulc.ai Feedback (~60 min)
**Resources:** Grinds360 essay notes; Pulc.ai; examinations.ie

1. Re-read phrase bank (5 min); no content notes
2. Write full Ceapadóireacht — timed, 40 min, no notes; 500–600 words; use formula from memory
3. Submit to Pulc.ai; read feedback — note saibhreas na Gaeilge flags, grammar penalties, structure comments
4. Pull marking scheme from examinations.ie; compare approach against what scheme rewards

**Target:** Concrete feedback on vocabulary, grammar, structure; know what to fix for S6

### Session 3 — Reading Comprehension: Format + Timing (~55 min)
**Resources:** Grinds360 (Léamhthuiscint section and any reading comp masterclass); Studyclix Plus; examinations.ie

1. Read Grinds360 Léamhthuiscint section and watch relevant masterclass — focus on: the different question types across the comprehension, what the longer written-response questions are actually asking, and how marks are allocated
2. Build timing plan: budget time per question based on mark allocation before starting any past paper attempt
3. Do one full timed reading comprehension on Studyclix (A + B together, strict time limit); mark against scheme; note which question type cost most marks

**Target:** Timing system built; question types mapped; first marked attempt done

### Session 4 — Prose Exam Technique (~50 min)
**Resources:** Pointsgame.ie; Studyclix Plus; examinations.ie; Grinds360 prose notes

1. Use Pointsgame.ie to identify 2 highest-frequency prose texts of your 6; focus on those
2. Attempt past paper prose questions on Studyclix without consulting notes first — forces active recall
3. Compare against examinations.ie scheme; then open Grinds360 notes to check what was missed (attempt → scheme → notes is the correct order)

**Target:** Know what prose answers reward; 2 texts attempted under exam conditions against scheme

### Session 5 — Poetry Exam Technique (~50 min)
**Resources:** Grinds360 (poetry technique section, Dínit an Bhróin module); Studyclix Plus; examinations.ie

1. Review Grinds360 poetry exam technique section — question types: an téama, mothúchán sa dán, an líne is fearr leat, do mhothú pearsanta
2. Attempt past paper questions on Dínit an Bhróin (Máirtín Ó Díreáin) via Studyclix using Grinds360 module as reference
3. Repeat for Úirchill an Chreagáin (Cathal Buí Mac Giolla Ghunna) using teacher notes + Grinds360 poetry technique section
4. Compare both against examinations.ie marking schemes; note what A-band answers do that yours didn't

**Target:** Both poems attempted under exam conditions; question-type patterns identified for each

### Session 6 — Second Essay + Reading Comp Consolidation (~60 min)
**Resources:** Grinds360; Pulc.ai; Studyclix Plus; examinations.ie

1. Write second timed essay on second-priority topic — 40 min, no notes; apply all S2 Pulc.ai feedback; submit to Pulc.ai again
2. Do second timed reading comprehension on Studyclix (different year); apply S3 timing strategy; full completion is the target
3. Review both schemes; write down top 3 mark sources still being missed — first Irish priorities for September

**Target:** Two graded essays and two timed reading comps completed; measurable summer progress logged
